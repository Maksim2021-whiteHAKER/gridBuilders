# GridBuilders

# **Сетевые создатели** - платформа для совместного 3D-строительства в реальном времени.

## 🔥 Возможности
- 🌐 Кроссплатфоменость: работайте на ПК и телефоне
   * ПК - полное действие, написание кода, моделинг, изменение ландшафта, экспорт\импорт.
   * Сенсорные устройства - передвижение объектов, создание объектов, импорт объектов.
- 👥 Кооперативное решение в реальном времени: строительство вместе с друзьями
- 📩 экспорт: в Roblox studio (.rbxm), GLTF, OBJ
- 👀 Понятный интерфейс с сеткой (grid-based building)

## 👩‍💻 Технологический стек
- **Frontend:** REACT + TYPESCRIPT + THREE.JS (REACT THREE fiber)
- **Backend:** Node.js + Express + Socket.io ()
- **State:** Zustand (хз)
- **Deployment:** (пока ищу)
