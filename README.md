# GridBuilders 🏗️

**Сетевые строители** — платформа для совместного 3D-строительства в реальном времени.

## 🎯 Какую проблему решает

В командной разработке Roblox-игр часто возникает ситуация: один разработчик на ПК, а другой — на телефоне. Мобильный Roblox Studio урезан, и полноценно строить карту вместе невозможно.

**GridBuilders** позволяет строить карты и модели вместе в реальном времени с любого устройства и экспортировать результат в Roblox Studio.

**Для кого:**
- 🛠 Разработчики Roblox-игр
- 🎨 3D-художники и дизайнеры уровней
- 👥 Команды без доступа к ПК для всех участников

## 🎮 Как использовать

1. **Создайте проект** — зарегистрируйтесь и создайте новую сцену
2. **Пригласите друга** — поделитесь ссылкой на проект
3. **Стройте вместе** — добавляйте объекты, перемещайте их в реальном времени
4. **Экспортируйте** — скачайте результат в .rbxm для Roblox Studio или .gltf для других движков

## 🔥 Возможности

- 🌐 **Кроссплатформенность:** работайте на ПК и телефоне
  - **ПК** — полный функционал: написание кода, моделинг, изменение ландшафта, экспорт/импорт
  - **Сенсорные устройства** — передвижение объектов, создание объектов, импорт моделей
- 👥 **Кооперативное строительство в реальном времени:** стройте вместе с друзьями
- 📩 **Экспорт** в Roblox Studio (.rbxm), GLTF, OBJ
- 👀 **Понятный интерфейс** с сеткой (grid-based building)

## 👩‍💻 Технологический стек

- **Frontend:** React + TypeScript + Three.js (React Three Fiber)
- **Backend:** Supabase (Realtime + PostgreSQL + Auth)
- **Хостинг:** Vercel
- **Синхронизация:** Supabase Realtime (WebSocket)
